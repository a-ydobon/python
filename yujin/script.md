# Python

## Python - Data
### 기본형
- 숫자형: integer&boolean, float
> - integer: 정수, 인티저
> - boolean: 불리안, 불, 등으로 불림 
             True, False를 나타냄. 숫자로 치환해 보면 False는 0 으로 대신 나타낼 수 있으며, True 는 0이 아닌 다른 정수를 통해 표현 가능.
- 문자형: string
### 묶음형
| 묶음 종류| 표현 예시| 순서 존재 여부 | 편집 가능 여부 |
| -------|-------| -------------|---------------|
| list | [1,2,3]  | O |O |
| tuple| (1,2,3) |  O|X|
| set | {1,2,3} |    X |O|
| dictionary | {'a':1,'b':2,'c':3}|    X|O|
